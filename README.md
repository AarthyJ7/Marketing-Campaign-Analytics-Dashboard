# Marketing Campaign Analytics Dashboard

Interactive marketing analytics dashboard built in Looker Studio using GA4 e-commerce data. Analyzes traffic acquisition, audience engagement, conversion funnel performance, and campaign effectiveness through KPI scorecards, dynamic filters, and executive reporting views.

* Calculated Fields
* Interactive Filters
